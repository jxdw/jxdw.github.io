背景

这次直接安装docker1.12。查了资料，docker1.12的最显著特性就是内置了docker自身的swarm调度。

docker1.12 实操

先看docker help和docker.hub.com。

docker login

命令作用:登录私有镜像仓库

输入账号和密码，即可提示登录成功。

docker tag

命令作用:对docker镜像打标签
命令使用:

docker pull

这个命令不需要多说了，从中央仓库或者私有仓库拉取镜像

docker push
